# Security Policy

## Supported versions

Only the latest tagged release of `alert-forwarder` receives security updates.

## Reporting a vulnerability

Please report vulnerabilities privately via GitHub's
[private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability)
on this repository.

You should receive an acknowledgement within 7 days. There is no bug bounty.
Do not open public issues or pull requests for security-sensitive reports.
